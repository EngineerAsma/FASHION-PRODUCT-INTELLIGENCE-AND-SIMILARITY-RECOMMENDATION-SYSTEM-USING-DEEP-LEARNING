PROBLEM STATEMENT:

Modern e-commerce platforms store product information in complex, deeply nested JSON formats. Extracting insights, understanding catalog structure, and building intelligent recommendation systems from such data is a significant engineering challenge.
The objective of this project is to build a Fashion Product Intelligence System that:
Converts complex JSON product data into a structured format
Performs detailed Exploratory Data Analysis (EDA)
Identifies trends across gender, season, usage, and color
Selects a single category (Men / Women / Kids)
Download and build the image dataset 
Builds a Deep Learning–based similarity recommendation system using product images
This project mirrors real-world data pipelines used in large e-commerce platforms.

PROJECT FLOW:
step 1: json ingestion and flattening
step 2: data cleaning and preprocessing
step 3: EDA
step 4: category selection for modelling
step 5: image  dataset for preparation
step 6: image classification model development
step 7: image embedding extraction
step 8: similarity computation and recommendation engine
